# Archive of the Awesome Generative AI List [![Awesome](https://awesome.re/badge-flat.svg)](https://awesome.re)

These projects are not active anymore in generative AI.

- Kite - Add powered code completions to your code editor, giving developers superpowers. [Code](https://github.com/kiteco).
- Pygma - Turn Figma designs into high-quality code.
- Coqui - Dedicated to open speech technology and to serving as the hub where speech researchers, developers, and practitioners congregate.
- Contenda - Create the content your audience wants, from content you've already made.
- LAIKA - LAIKA trains an artificial intelligence on your own writing to create a personalised creative partner-in-crime.
- ChatGPT for Search Engines - Display ChatGPT response alongside Google, Bing, and DuckDuckGo search results.
- Galactica - A large language model for science. Can summarize academic literature, solve math problems, generate Wiki articles, write scientific code, annotate molecules and proteins, and more. [Code](https://github.com/paperswithcode/galai).
- Synthical - AI-powered collaborative research environment.
- Debuild - AI-powered low-code tool for web apps.
- MutableAI - AI Accelerated Software Development.
